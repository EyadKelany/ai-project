Core Topics

Fundamental Concepts

Deep dive into each foundational area with practical resources.

Supervised Learning

Learning from labeled examples (X, y) to predict outputs for new inputs.

  • Classification: Logistic regression, SVM, Random Forest, XGBoost, Neural nets
  • Regression: Linear, Ridge/Lasso, Gradient Boosting, Quantile regression
  • Key metrics: Accuracy, F1, ROC-AUC, Precision/Recall, MSE, MAE, R²
View Q&As sklearn Guide

Unsupervised Learning

Discovering patterns in data without labels — clustering, dimensionality reduction, density estimation.

  • Clustering: K-Means, DBSCAN, HDBSCAN, Hierarchical, Gaussian Mixture
  • Dimensionality Reduction: PCA, t-SNE, UMAP, Autoencoders
  • Anomaly Detection: Isolation Forest, One-Class SVM, Elliptic Envelope
View Q&As sklearn Guide

Neural Networks & Backprop

Universal function approximators trained via gradient descent and the chain rule.

  • Architecture: Layers, neurons, weights, biases, activations (ReLU, GELU, Swish)
  • Training: Forward pass → Loss → Backward pass (autograd) → Optimizer step (Adam, AdamW)
  • Regularization: Dropout, weight decay, batch/layer norm, data augmentation, early stopping
View Q&As PyTorch Tutorial

Convolutional Neural Networks

Translation-equivariant architectures for grid-like data (images, audio spectrograms, time series).

  • Building blocks: Conv2D, pooling, stride, dilation, depthwise separable, groups
  • Modern architectures: ResNet, EfficientNet, ConvNeXt, NFNet, ViT hybrids
  • Applications: Classification, detection (YOLO, Faster R-CNN), segmentation (U-Net, DeepLab)
View Q&As Annotated Implementations

Sequence Models (RNN/LSTM/GRU)

Processing sequential data with recurrent connections and gating mechanisms.

  • Vanilla RNN: Simple recurrence, vanishing gradient problem
  • LSTM: Input/forget/output gates, cell state, constant error carousel
  • GRU: Simplified gating (reset/update), fewer params, similar performance
View Q&As Understanding LSTMs

Attention & Transformers

The architecture that enabled modern LLMs — parallelizable, global context, compositional.

  • Self-Attention: Q·Kᵀ/√d scaling, softmax, weighted value aggregation
  • Multi-Head: Parallel attention heads → concat → project
  • Architecture: Pre-LN, residual, FFN (SwiGLU/GeLU), RoPE/ALiBi positions
View Q&As Annotated Transformer

Embeddings & Representations

Dense vectors where semantic similarity = geometric proximity. The universal language of modern AI.

  • Static: Word2Vec, GloVe, FastText — context-independent
  • Contextual: BERT, GPT, E5, BGE — token meaning depends on context
  • Sentence/Multimodal: SBERT, CLIP, ImageBind — cross-modal alignment
View Q&As MTEB Leaderboard

Optimization & Training Tricks

Making deep learning actually converge — the engineering behind the science.

  • Optimizers: AdamW, Lion, Sophia, Muon — adaptive learning rates, weight decay decoupling
  • Schedules: Cosine decay, warmup, linear decay, constant + cooldown
  • Mixed Precision: FP16/BF16, gradient scaling, FSDP/ZeRO for distributed training
View Q&As Lion Optimizer

Evaluation & Model Selection

Rigorous assessment prevents overfitting to benchmarks and ensures real-world performance.

  • Validation: Train/val/test splits, k-fold CV, stratified, group-aware, time-series splits
  • Metrics per task: Classification (F1, AUC, calibration), Regression (MAE, RMSE, quantile), Ranking (NDCG, MRR)
  • Statistical significance: Bootstrap CI, paired t-test, McNemar, permutation tests
